# Брзи старт
Аутоматизовано генерисање пореске пријаве ППДГ-3Р (Капитална добит) и ПП ОПО (Приходи од капитала) за кориснике Interactive Brokers у Србији.
Програм аутоматски преузима податке о трансакцијама и креира спреман XML фајл за отпремање, конвертујући све цене у динаре (РСД).

1. [Инсталирајте ibkr-porez ↗](installation.md)

2. [Конфигурација (config) ↗](usage.md#config)

3. [Увоз историјских података (import) ↗](usage.md/#import)

    > ⚠️ **Увоз је потребан само ако имате више од годину дана историје трансакција у Interactive Brokers.** Flex Query омогућава преузимање података за не више од последње године, тако да старији подаци морају бити учитани из [пуног извоза у CSV фајл ↗](ibkr.md/#извоз-пуне-историје-за-импорт-команду).

=== "Брзо креирати одређену пријаву"

    Ако желите брзо да креирате одређену пријаву.

    [Преузимање најновијих података (fetch) ↗](usage.md/#преузимање-података-fetch)

    [Креирање извештаја (report) ↗](usage.md/#генерисање-пореског-извештаја-report)

    Отпремите креирани XML на портал **еПорези** (секција ППДГ-3Р).
        ![ППДГ-3Р](images/ppdg-3r.png)

=== "Аутоматско креирање пријава"

    Ако желите аутоматски да примате све потребне пријаве и пратите њихов статус (поднета, плаћена).

    [Преузимање најновијих података и креирање пријава (sync) ↗](usage.md/#синхронизација-података-и-креирање-пријава-sync)

    [Управљање пријавама ↗](usage.md/#_2)
