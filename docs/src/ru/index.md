# ibkr-porez

Автоматическая генерация налогового отчета ППДГ-3Р (Капитальный доход) для пользователей Interactive Brokers в Сербии.
Программа автоматически получает данные о транзакциях и создает готовый XML-файл для загрузки, конвертируя все цены в динары (RSD).

![ППДГ-3Р](images/ppdg-3r.png)

[Установите ibkr-porez](installation.md)

[Настройка](setup_ibkr.md): Сохраните учетные данные Interactive Brokers Flex Query и данные налогоплательщика.
    ```bash
    ibkr-porez config
    ```

[Загрузка данных](usage.md/#get): Скачайте историю транзакций из Interactive Brokers и официальные курсы валют Народного банка Сербии.
    ```bash
    ibkr-porez get
    ```

[Создание отчета](usage.md/#report): Сгенерируйте XML-файл ППДГ-3Р.
    ```bash
    ibkr-porez report
    ```

> Просто загрузите созданный XML на портал **ePorezi** (раздел ППДГ-3Р).
