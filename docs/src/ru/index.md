# Быстрый старт
Автоматическая генерация налогового отчета ППДГ-3Р (Налог на прирост капитала) и ПП ОПО (Налог на доходы от капитала) для пользователей Interactive Brokers в Сербии.
Программа автоматически получает данные о транзакциях и создает готовый XML-файл для загрузки, конвертируя все цены в динары (RSD).

[Установите ibkr-porez ↗](installation.md)

Далее вы можете использовать как графический интерфейс (запускается как `ibkr-porez` без параметров) так и командную строку - см. ниже.
Графический интерфейс и командная строка используют одну и ту же БД.

   > ⚠️ При запущенном графическом интерфейсе не используйте командную строку,
   > при одновременной работе они могут внести ошибки в БД.

В графическом интерфейсе настройте свои данные (кнопка config) и для обновления данных и создания деклараций просто нажимайте кнопку Sync.

Если вы хотите все делать через командную строку то далее надо:

- [Настройка (config) ↗](usage.md#config)
- [Импорт исторических данных (import) ↗](usage.md/#import)

    > ⚠️ **Импорт необходим, только если у вас больше года истории транзакций в Interactive Brokers.** Flex Query позволяет загрузить данные не более чем за последний год, поэтому более старые данных необходимо загрузить из [полного экспорта в CSV файл ↗](ibkr.md/#import).

=== "Быстро создать нужную декларацию"

    Если вы хотите быстро создать конкретную декларацию.

    [Загрузка последних данных (fetch) ↗](usage.md/#fetch)

    [Создание отчета (report) ↗](usage.md/#report)

    Загрузите созданный XML на портал **ePorezi** (раздел ППДГ-3Р).
        ![ППДГ-3Р](images/ppdg-3r.png)
